Tugas 01

Nama Tag dan fungsinya:

  • <Hn>..</Hn> : Heading, untuk membuat judul
  • <p>..</p> : Paragraf, untuk membentuk paragraf
  • <BR>t : Break, untuk memaksa ganti baris
  • <B>..</B> : Bold, untuk mencetak tebal
  • <I>..</I> : Italic, untuk mencetak miring
  • <U>..</U> : Underline, untuk mencetak garis bawah
  • <FONT>..</FONT> : Font, untuk memanipulasi huruf

Catatan:

 Beberapa tag bisa digunakan secara bersamaan, misalnya untuk keperluan menampilkan kata seperti ini

Juga bisa menggunakan beberapa parameter
